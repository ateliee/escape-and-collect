# ゲーム仕様書 (Escape And Collect)

## 1. ゲームの目的
敵キャラクターから逃げつつ、ステージ上にランダムで出現するコインを集めてスコアを稼ぐこと。敵に接触すると即座にゲームオーバーとなる。

## 2. 画面遷移フロー
1. **タイトル画面 (`TitleScreen`)**
   - 「START」ボタンをクリックしてメインゲームへ。
2. **メインゲーム (`World`)**
   - プレイヤーを操作し、コインを集める。
   - プレイ中はマウスカーソルが非表示になり、カメラ操作に割り当てられる。
3. **ゲームオーバー画面 (`GameOverScreen`)**
   - 獲得したスコアを表示。
   - 「RETRY」ボタンをクリックして再度メインゲームへ。

## 3. キャラクター仕様

### 3.1 プレイヤー (`Player`)
- **視点**: 3人称視点（サードパーソン / `SpringArm3D` を使用したカメラ追従）。
- **移動**: 方向キーまたはWASDによる移動。カメラの向いている方向を基準に移動方向が計算される。
- **ジャンプ**: `Space` キー。
- **ダッシュ**: `Shift` キーで一定時間移動速度が上昇。終了後、再使用までのクールダウンタイムが存在する。
- **やられ判定**: 敵のコライダーと接触した瞬間に `die()` 関数が呼ばれ、ゲームオーバーとなる。

### 3.2 敵 (`Enemy`)
- **行動**: `NavigationAgent3D` を使用し、常にプレイヤーの現在位置をターゲットとしてナビゲーションメッシュ（床）上を最短経路で追跡する。
- **スポーン**: 一定時間ごとにランダムな位置（プレイヤーから一定距離以上離れた場所）にスポーンする。ゲーム内の最大存在数（上限）が設定されている。

## 4. アイテム仕様

### 4.1 コイン (`Coin`)
- 取得するとスコアが `+1` 加算される。
- メインゲーム開始時に複数個が初期配置され、その後も一定時間ごとに新しいコインがランダムな位置に追加される。
- プレイヤーとの接触（`Area3D` の `body_entered`）で取得判定を行い、スコア加算のシグナルを発行したのち自身を削除する。

## 5. デバッグ・その他の機能
- **強制終了**: プレイ中（マウスがキャプチャされた状態）に `Esc` キーを押すことで、即座にアプリケーションを終了できる。
- **利用アセット**: プレイヤー、敵、コイン、およびステージのモデリングには「Kenney Starter Kit 3D Platformer (CC0)」を使用している。
