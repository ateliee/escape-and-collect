# Escape And Collect (Godot 3D Game)

シンプルな3D逃走＆コイン収集ゲームです。
Godot Engine 4を使用して作成されています。

## 概要
棒人間（敵キャラクター）から逃げながら、フィールド上のコインを集めてハイスコアを目指すサードパーソン視点（TPS）のアクションゲームです。

## 操作方法
- **移動**: `W` `A` `S` `D` キー または 矢印キー（`↑` `↓` `←` `→`）
- **ジャンプ**: `Space` キー
- **ダッシュ**: `Shift` キー
- **視点移動**: マウス操作
- **ゲーム終了（デバッグ時）**: `Esc` キー
